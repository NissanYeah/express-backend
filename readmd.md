# README

# 環境設定

- server要安裝nvm以切換node版本
- 安裝nodejs
- 安裝nodemonjs 用來快速變化
- 用kill port去刪除佔用的port

# 套件

- npm install body-parser
- npm install mongoose
- npm install kill-port --save-dev  去kill port
- npm install nodemon
- npm install express-session
- npm install passport passport-local
